# Troubleshooting

Common issues and solutions.

## Installation Issues

### Permission Errors
```bash
pip install --user he2plus
```

### Network Issues
- Check internet connection
- Try again later
- Use different mirror

## Profile Installation

### Out of Disk Space
- Free up space
- Install one profile at a time

### Slow Installation
- Check network speed
- Be patient (can take 10-20 minutes)
